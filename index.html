Welcome to the CS633 Automated Testing and Code Quality Page

Automated testing and code quality tools have become the gold standard within Agile / Scrum / Extreme Programming (XP) teams. By using automated tests including unit, integration, user interface; combined with code quality analysis tools, SonarQube, Veracode - these teams can develop high quality (confidence) software and fix defects close to the point of their origin. These techniques also facilitate the trust to refactor code, because if something breaks, the regression tests find defects almost immediately. By working though these lessons you too will become convinced of the power and need of these techniques.

In lesson one you will use the JavaScript framework Mocha and the BDD / TDD assertion library Chai to execute tests (directly in the browser) for the classic game rock-paper-scissors.
